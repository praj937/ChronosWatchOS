# ChronosWatchOS

A modern smartwatch firmware for the ESP32-S3 using ChronosESP32.

## Hardware

- ESP32-S3-WROOM-1-N16R8
- SSD1306 128×64 OLED (I2C)
- Android Phone running Chronos

## Features

- Google Maps Navigation
- Notifications
- Incoming Calls
- Phone Battery
- Watch Battery
- Bluetooth Status
- Music Controls
- Weather
- AI Assistant (planned)
